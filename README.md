# Flow Board

Projeto simples de quadro Kanban feito com HTML, CSS e JavaScript puro, com visual inspirado em ferramentas como o Jira.

## Sobre o projeto

O Flow Board simula um quadro de tarefas para acompanhamento de sprint. A interface permite visualizar cards em colunas, filtrar tarefas e gerenciar itens de forma prática, sem depender de backend.

## Funcionalidades

- Visualização de tarefas em colunas de status
- Criação de novas tarefas
- Edição e exclusão de tarefas
- Filtro por prioridade
- Busca por título e descrição
- Resumo rápido com indicadores do quadro
- Movimentação de cards entre colunas com drag and drop

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Como executar

Como este é um projeto frontend estático, basta abrir o arquivo `index.html` no navegador.

Se preferir, também é possível rodar com uma extensão como Live Server no VS Code.

## Observações

- Os dados das tarefas ficam armazenados apenas em memória
- Ao recarregar a página, as alterações são perdidas
